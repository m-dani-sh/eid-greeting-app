# Eid Mubarak Wish Generator 🌙

A beautiful, interactive web application for creating personalized Eid wishes with stunning animations and effects.

## Features

- **Personalized Wishes**: Generate custom Eid greetings by entering a name
- **Stunning Animations**: Fireworks, balloons, confetti, and floating particles
- **Share Functionality**: Share wishes via WhatsApp or copy link to clipboard
- **Responsive Design**: Works perfectly on all devices
- **Firebase Integration**: Stores wish data in the cloud
- **Beautiful UI**: Glass morphism effects with moon theme

## Files Structure

- `eid-wisher.html` - Main HTML file with the wish generator interface
- `script.js` - JavaScript functionality for animations, wish generation, and sharing
- `style.css` - Styling with glass morphism effects and animations
- `firebase.js` - Firebase configuration for data storage

## How to Use

1. Open `eid-wisher.html` in your web browser
2. Enter a name in the input field
3. Click "Generate Wish 🌙" to create a personalized greeting
4. Share the generated link with friends and family
5. Click "Open Your Eid Surprise" for special animations

## Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Styling**: CSS animations, glass morphism design
- **Backend**: Firebase Firestore for data storage
- **Fonts**: Google Fonts (Cinzel Decorative, Cormorant Garamond, DM Sans)

## Firebase Setup

The app uses Firebase for storing wish data. Update the Firebase configuration in `firebase.js` with your own project credentials if needed.

## Sharing

- **WhatsApp**: Direct share to WhatsApp with custom message
- **Copy Link**: Copy the personalized wish link to clipboard
- **URL Parameters**: Name is passed via URL for easy sharing
